## 🚀 Getting Started

### 📦 Dependencies

- Hugo v0.115.1+
- Node v18+
- Npm v9.5+
- Tailwindcss v3+

### 👉 Project Setup

```
npm run project-setup
```

### 👉 Install Dependencies

```
npm install
```

### 👉 Development Command

```
npm run dev
```

### 👉 Build Command

```
npm run build
```
